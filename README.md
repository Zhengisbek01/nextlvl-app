# NextLVL 🛡️

**Level up your life** — персональный трекер роста с ИИ-советником Aza

## Функции
- 🎯 Трекинг привычек с геймификацией и XP
- ✅ Управление задачами с приоритетами
- 💰 Финансовый учёт в ₸
- ⬡ 10 уровней прокачки
- 🤖 ИИ-советник Aza (Claude AI)
- 🌐 Три языка: RU / KK / EN
- 🎨 4 темы: Dark Gamer / Dark Purple / Gold Premium / iOS Blue

## Стек
- React 18 + Vite
- Recharts
- Claude API (Anthropic)
- PWA / Vercel

## Запуск
```bash
npm install
npm run dev
```

## Деплой
Подключи репозиторий на vercel.com — деплой автоматический.
